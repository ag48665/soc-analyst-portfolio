# Sysmon Process Creation Monitoring

## Objective

Monitor process creation events using Sysmon.

## Event Details

### Event ID 1

Process Creation Event

### Observed Process

- Image: C:\Windows\System32\conhost.exe
- Parent Process: C:\Windows\System32\cmd.exe
- User: Agata

## Analysis

Sysmon detected the creation of a new process generated by cmd.exe.

The event captured:

- Process Name
- Parent Process
- Process ID
- User Context
- SHA256 Hash

## Security Relevance

Process creation monitoring helps detect:

- Malware execution
- Command execution
- PowerShell abuse
- Living-off-the-Land techniques
